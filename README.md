# Command-index（命令行博客首页）V1.05
Wordpress 的 一款仿命令行的博客首页插件！
### 功能介绍
* 命令可以列出文章和分类，支持翻页、分类列出
* 支持自定义一些命令
* 可以自主切换 命令行/网页 访问首页
